# VoiceMate Frontend

음성 클론 기반 크로스 언어 소개팅 앱의 Expo (React Native + TypeScript) 프론트엔드.

## 하네스: VoiceMate FE Build

**목표:** BE API 스펙 분석 → FE 코드 구현 → FE↔BE 통합 검증의 3단계 파이프라인으로 정확한 프론트엔드 코드를 생성한다.

**트리거:** 화면 구현, 컴포넌트 개발, API 연동, 기능 추가/수정 등 프론트엔드 개발 작업 요청 시 `voicemate-build` 스킬을 사용하라. 단순 질문이나 설정 변경은 직접 응답 가능.
